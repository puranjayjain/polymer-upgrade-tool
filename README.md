# polymer-upgrade-tool
a command line tool that helps you upgrade your polymer project from 1.x to pure 2.x class style elements

# installation

Use npm

```
npm i --save-dev polymer-upgrade-tool
```

or use yarn

```
yarn add -D polymer-upgrade-tool
```
